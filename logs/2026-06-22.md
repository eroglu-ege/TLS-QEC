## UNDERSTOOD

### T2 sweep — threshold region results
  g/gamma_t=0.100: Solomon ALWAYS valid (D<0.05 at all T2/T1)
  g/gamma_t=0.216: Solomon ALWAYS valid (just below significance threshold)
  g/gamma_t=0.500: valid only for T2/T1 < 0.559
  g/gamma_t=0.860: valid only for T2/T1 < 0.231  (sign reversal point)
  g/gamma_t=1.000: valid only for T2/T1 < 0.166
  g/gamma_t=1.950: NEVER valid — hard boundary
  g/gamma_t=3.000: NEVER valid
  → Hard boundary at g/gamma_t ~ 2: beyond this no dephasing saves Solomon
  → ISE drops ~99% as T2 goes 2T1→0.1T1 across all g
  → Coherence suppressed ~91% universally but baseline coherence grows with g

### T2 enters Solomon via exchange rate linewidth
  Gamma = 2g^2 * gamma_t_eff / (Delta^2 + gamma_t_eff^2)
  gamma_t_eff = gamma_t*(2*n_th+1) + gamma_phi
  → Pure dephasing broadens TLS linewidth → reduces peak Gamma
  → Shorter T2 = faster coherence death = Solomon more valid
  → Breakdown condition: g * T2_eff >> 1 (many coherent cycles per T2)

### Trace distance as rigorous significance metric
  D(rho_L, rho_S) = (1/2) Tr|rho_L - rho_S|, bounded [0,1]
  Convention: D < 0.01 negligible, D > 0.05 substantial
  From fine scan: D>0.05 first at g/gamma_t = 0.216
  Single trajectory (g=0.1): D peaks at 0.726 — 73% distinguishable

### Why Solomon overshoots Lindblad at intermediate g
  Solomon: monotonic diffusion, pushes population fast
  Lindblad: coherent Rabi — population retreats after each half-cycle
  At g/gamma_t ~ 0.5-1.95: Solomon's push faster than Lindblad's
  oscillation → overshoot. Past ~1.95: Lindblad's Rabi transfer wins.

### make_params() workflow

  T2=None → T2=2*T1 (no dephasing, Lindblad limit)
  T2=100  → gamma_phi computed automatically
  Files tagged: nth0.1000_T2q200_T2t400 etc.
