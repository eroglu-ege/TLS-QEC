## UNDERSTOOD
- g=gamma_t threshold physically: crossing point of two timescales
    T_rabi = pi/g           (coherent exchange time)
    1/gamma_t                (TLS decoherence time)
  g<<gamma_t: TLS dephases before exchange completes -> Solomon exact
  g>>gamma_t: many coherent cycles survive -> Solomon fails
- "Breakdown" needs TWO numbers, not one:
    sensitivity onset  = first detectable departure (10x noise floor)
    significance level = first point exceeding a physically meaningful
                          cutoff (trace distance D>0.05, standard QI convention)
- Trace distance D = (1/2)Tr|rho_L - rho_S| is bounded [0,1], operationally
  meaningful (max probability of distinguishing states via optimal measurement)
- Single trajectory test: D starts at 0 (identical IC), peaks ~0.52 during
  transient, returns to 0 at thermal steady state — physically sane
- Baseline estimation is sensitive to window choice — must exclude points
  where real signal has already begun, or baseline gets contaminated
